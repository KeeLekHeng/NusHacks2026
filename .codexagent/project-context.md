# Project Context

This is a hackathon project designed to ship a polished MVP fast in one day.

Primary stack:
- Next.js + TypeScript + Tailwind frontend
- FastAPI backend

Agent layers:
- TinyFish is the web execution layer (website automation/extraction)
- OpenAI is the reasoning and summarization layer

Priority order:
1. Working end-to-end demo
2. Clean UX
3. Stable API contracts
4. Simple, impressive agent flow

Judging context:
- Project should showcase autonomous web browsing/extraction on live sites.
- Judges will likely reward real-time usefulness, technical ambition, and handling of messy web data.
- The product should feel credible, fast to understand, and impressive in a short live demo.

Do not add unless explicitly requested:
- Auth
- Databases
- Queues
- Broad refactors
- Unnecessary abstractions
